Item Details

This API call returns the metadata associated with a Pin, the primary data object within the Historypin repository. This data structure includes everything that we store about Pins. This call also includes enough information to build the full user interface for a Pin, which includes some information about the Collections which this Pin is associated with (or "Repinned" to).

URL

http://www.historypin.org/en/api/pin/get.json{?parameters}

Parameters

  • id (integer, required) - A value representing the Item Identification Number (ID)
